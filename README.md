# Products App

Учебный проект интернет-магазина на Vanilla JavaScript.

## О проекте

Приложение получает товары из FakeStore API и позволяет искать, фильтровать и просматривать товары в удобном интерфейсе.

## Возможности

* Загрузка товаров через FakeStore API
* Поиск товаров
* Фильтрация по категориям
* Фильтрация по цене
* Сортировка товаров
* Модальное окно с подробной информацией о товаре
* Responsive дизайн для мобильных устройств

## Технологии

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* FakeStore API
* Git & GitHub

## Запуск проекта

1. Склонировать репозиторий
2. Открыть файл `index.html`
